Creating Dataset Summaries:

The summary tool allows you to calculate descriptive statistics for selected variables in your dataset. You can summarize variables across one or more grouping columns.

How to use the summary tool:

  • Select one or more grouping variables under Variables to group your data.
  • Select one or more numeric variables under Which Variables to summarize.
  • Select one or more summary functions under Which functions should be applied.
  • Click summarize to create a summary table.
  • Click summarize plot to create a graphical summary.

Available summary functions:

  • Mean: Calculates the average value.
  • Median: Calculates the middle value.
  • SD: Calculates the standard deviation.

Grouping behavior:

If one or more grouping variables are selected, OpenStats calculates the selected summary functions separately for each group combination.

Example:

group treatment value
A Control 10.2
A Control 12.1
B Treated 18.4
B Treated 20.0

If group and treatment are selected as grouping variables, and value is selected as the variable to summarize, OpenStats will calculate the selected functions separately for each group-treatment combination.

Important notes:

  • You must select at least one grouping variable.
  • You must select at least one variable to summarize.
  • You must select at least one summary function.
  • The selected summary functions should be meaningful for the selected variables.
